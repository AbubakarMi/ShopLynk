## üìã Quick Reference

### Start Development Servers

**Web:**
```bash
cd web
npm run dev
# Open: http://localhost:5173
```

**Mobile:**
```bash
cd mobile
npx expo start
# Press 'i' for iOS or 'a' for Android
```

### Admin Panel Access

**Web:**
- Direct: http://localhost:5173/admin/dashboard
- From Sign-In: Click "üîê Access Admin Panel"

**Mobile:**
- From Sign-In: Tap "Access Admin Panel ‚Üí"
- Tap hamburger menu (‚ò∞) to open sidebar

### Key Colors

- **Primary:** `#3B5BDB` (Blue)
- **Accent:** `#00C896` (Teal/Green)
- **Gradient:** `from-[#3B5BDB] to-[#00C896]`
